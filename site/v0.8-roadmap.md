# Historical v0.8 Scope Notes

This file is kept as a historical note for the pre-v1.0 site track. It is **not** the current product roadmap.

For current release positioning, use:

- `docs/roadmap.md`
- `README.md`
- `README.zh-CN.md`

## What v0.8 contributed

The v0.8 track moved the public site beyond a basic LaTeX checker landing page and added the defense/showcase surfaces that are now part of the v1.0 public story:

- bounded defense-prep artifact generators under `17-defense-pack/`
- artifact gallery page under `site/artifact-gallery.html`
- docs home under `site/docs-home.html`
- scenario entry page under `site/scenario-entry.html`
- advisor handoff and submission-prep scenario pages

## Boundaries that remain true in v1.0

These are still intentionally out of scope:

- full GUI or Web editor
- perfect general LaTeX-to-Word conversion
- automatic understanding of advisor natural-language intent
- multi-user real-time collaboration platform
- replacement for Overleaf / TeXstudio / TeX toolchains
- one-click thesis submission
- automatic final defense PPT generation

## Current principle

v1.0 continues the same bounded philosophy:

1. every workflow has explicit inputs, outputs, and limits
2. unsupported constructs and degradation risks are reported explicitly
3. high-judgement decisions remain human-owned
4. public copy must match real scripts, artifacts, and rule-pack behavior
