LaTeX 学位论文自查

引用断了、格式混了、标点串了 LaTeX 不会替你报错

这些问题往往到提交当天才暴露。Thesis Skills 在提交前把引用、格式、语言问题系统查一遍，输出结构化报告。

01 / capabilities

四个核心检查模块

每项检查输出独立 JSON 报告，列出具体发现和位置。

01

引用完整性检查

对照 .bib 文件扫描 \cite{key}，列出缺失引用和孤立 bib 条目。

02

CJK 语言风格检查

标记中英文标点混用、弱语气词位置。逐条列出，附行号。

03

图表结构验证

找出缺 label 或 caption 的 figure/table 环境。逐条报告。

04

Word → LaTeX 迁移

章节拆分 + 引用格式转换 + 迁移检查报告。单次运行。

02 / facts

三个可核实的事实

零外部依赖 Python stdlib，不安装第三方包
GB/T 7714-2015 内置 thuthesis 规则集
JSON + exit code 结构化输出，可接入 CI
03 / architecture

运行方式

一条命令 python run_check_once.py 依次调用各检查模块，读取 YAML 规则配置，对你的 LaTeX 项目文件夹输出 JSON 报告。exit code 合约：0 pass / 1 findings / 2 config error。

04 / quick start

跑一次试试

python run_check_once.py --project-root "../thuthesis-v7.6.0" --rules tsinghua
