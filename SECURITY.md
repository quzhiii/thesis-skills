# Security Policy

## Supported Versions

This project is currently in staging (`0.x`). Security fixes are applied to the latest main branch only.

## Reporting a Vulnerability

- Please do not open public issues for potential security vulnerabilities.
- Report privately to maintainers with:
  - affected file/path
  - reproduction steps
  - impact assessment
  - suggested fix (if available)

## Scope Notes

- This project is a local thesis QA/checking toolkit.
- It does not require API keys by default.
- Do not commit personal thesis data, secrets, or private institutional materials to public repos.
